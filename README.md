# Sensorimotor-EEG-responses-to-cueing-in-PD

This repo contains the analysis MATLAB scripts used to preprocess the scalp EEG data described in the manuscript: 

>Ma L, Miura M, Talu I, Cross KA. Sensorimotor cortex activity during cued gait differs in Parkinson's freezers and non-freezers. Currently under review.

Under the `code` folder are the main scripts used to process the EEG data, along with supporting functions. This does not include any functions from external toolboxes such as EEGLAB, FieldTrip, FOOOF, or built-in MATLAB toolboxes.
