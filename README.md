# Sensorimotor-EEG-responses-to-cueing-in-PD

This repository contains MATLAB scripts for preprocessing scalp EEG and R scripts for performing statistical analyses on the kinematic and EEG measures detailed in the manuscript:

>Ma L, Miura M, Talu I, Cross KA. Sensorimotor cortex activity during cued gait differs in Parkinson's freezers and non-freezers. Currently under review.

READMEs for each processing pipeline are in their respective folders under `code/`.  Some scripts may require functions from MATLAB (toolbox-specific), external toolboxes (EEGLAB, FieldTrip, FOOOF), and R libraries.
